WEB DEVELOPMENT SKILLS

Learn HTML

HTML stands for HyperText Markup Language. HTML is the content of a Webpage.
HTML includes headings, paragraphs, horizontal rules, images, links, lists, etc.

Check out HTML tutorials on w3schools.com

Learn CSS

CSS stands for Cascading Style Sheets. CSS is the style of a web page.
CSS includes font styles, colour, background colour, borders, dimensions, layout, etc.

Check out CSS tutorials on w3schools.com

Learn Javascript

Javascript is the programming language of the web. Javascript helps to make webpages interactive and dynamic. Javascript Tasks includes:

  • Adding new HTML to the page
  • change the existing content / style
  • react to user events (mouse clicks, key presses, etc.)

Check out Javascript tutorials on Javascript.info
